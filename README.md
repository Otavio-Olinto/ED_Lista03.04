# ED_Lista03.04
## Binário: Recursiva

Este desafio consiste em criar uma aplicação, em Java, que receba um número natural e apresente sua forma em binário.

```
Requisitos do projeto:

1. Arquitetura MVC;
2. A entrada deve ser limitada para até 2.000;
3. A conversão deve ser realizada por uma função recursiva;
4. Declarar, via comentário, a condição de parada da função.
```
